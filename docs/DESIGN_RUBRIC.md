# 設計準則（taste-distiller 萃取，溫暖人文風）

改版與發布前逐條自我打分，**低於 3 不上線**。

| # | 準則 | Reject（避免） | Want（追求） |
|---|---|---|---|
| 1 | **Icon 與品牌** | emoji 當招牌/功能識別 | 統一線寬圓角的線性 SVG icon + 品牌標誌 |
| 2 | **色彩與情感** | 藍色漸層、科技冷色 | 暖橘 × 米白暖色系，公益柔和、被接住的安定感 |
| 3 | **版面節奏** | 一排等寬白卡的罐頭模板 | hero/內容/行動不同節奏 + 留白呼吸 + 插畫穿插 |
| 4 | **插畫氛圍** | 空泛填充 / AI 生成圖 / 通用 clipart | 一致風格向量插畫，貼近反詐情境、有同理心 |
| 5 | **可信來源露出** | 純文字提及、零連結 | 重點處放 165 / 打詐儀錶板真實連結卡 |

整體目標：讓人感覺「**有人用心做、暖、可信**」，而不是隨手生成的模板。
